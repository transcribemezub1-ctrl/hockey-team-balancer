## Fastest Way to Deploy (5 Minutes!)

### Option A: GitHub + Vercel (Recommended)

1. **Upload to GitHub**
   - Go to github.com and create account
   - Create new repository: `hockey-team-balancer`
   - Upload all the files I provided

2. **Deploy to Vercel**
   - Go to vercel.com
   - Click "Sign up with GitHub"
   - Click "Import Project"
   - Select your `hockey-team-balancer` repo
   - Click "Deploy"
   - Done! Get your URL

3. **Update Anytime**
   - Change files on GitHub
   - Vercel automatically redeploys
   - Your URL stays the same

### Option B: Quick CLI Deploy

```bash
npm install -g vercel
cd your-project-folder
vercel login
vercel
```

That's it! You'll get a live URL instantly.
